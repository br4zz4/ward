# Vaults Command

> TLDR: Add a `ward vaults` command that lists all configured vault paths from `.ward/config.yaml`.

**Status:** completed
**Created:** 2026-04-21
**Owner:** @oporpino

---

## Context

Users need a quick way to see which vault directories are configured for the current project without opening the config file manually.

## Objectives

- Add `ward vaults` command that prints each configured vault path (one per line)

## Changes

- `internal/cmd/vaults.go` — new command `NewVaultsCmd()` that reads `cfg.Vaults` and prints each path
- `cmd/ward/main.go` — register `cmd.NewVaultsCmd()`

## How to verify

```bash
ward vaults
# outputs each vault path from .ward/config.yaml, one per line
```
